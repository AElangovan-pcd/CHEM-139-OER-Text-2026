Chapter 7: Chemical Nomenclature

CHEM 139 — Open Educational Resource (2026 ed.) · CC BY 4.0

Chapter Opener: A Common Language

In the 1700s a chemist who wanted to write up an experiment had to decide whether to call sulfuric acid "oil of vitriol," call iron(III) chloride "butter of mars," or call sodium carbonate "soda ash." The names were colorful, often poetic, and almost completely useless to anyone trying to repeat the work. By the early 1900s, chemists had agreed on a systematic naming scheme — one in which the name of a compound tells you exactly which elements are in it and in what ratio.

This chapter teaches you that scheme. It is not glamorous material; it is the alphabet of chemistry, and you cannot do the rest of the course without it. By the end of the chapter you should be able to take a name like "calcium hydrogen phosphate" and write down Ca(HPO₄), or take a formula like Fe₂(SO₄)₃ and produce the name "iron(III) sulfate."

Chapter Learning Outcomes

  1. Classify any compound as Type I or Type II ionic, polyatomic-ion-containing ionic, binary molecular, or acid.
  2. Apply the correct naming convention to each class.
  3. Name and write formulas for binary ionic compounds with fixed- and variable-charge metals (Stock system).
  4. Name and write formulas for compounds containing polyatomic ions.
  5. Name binary molecular compounds using Greek prefixes.
  6. Name binary acids and oxyacids correctly given the formula, and produce the formula given the name.
  7. Use a systematic decision-tree procedure to name any covered compound.

7.1 Classification of Compounds for Nomenclature Purposes

Different classes of compounds use different naming rules. Before you can name a compound you must decide which class it belongs to. We use four classes in this chapter:

  • Binary ionic compounds. Two elements, one of which is a metal cation and the other a nonmetal anion. Subdivided into Type I (fixed-charge metal) and Type II (variable-charge metal).
  • Ionic compounds containing polyatomic ions. More than two elements, with one or more polyatomic ions present.
  • Binary molecular compounds. Two nonmetals (or two metalloids, or one nonmetal and a metalloid) covalently bonded.
  • Acids. Compounds beginning with H, when dissolved in water, that release H⁺ ions. Subdivided into binary acids (hydroacids: H + one nonmetal) and oxyacids (H + a polyatomic ion containing oxygen).

7.2 Types of Binary Ionic Compounds

Binary ionic compounds come in two types based on whether the metal cation has a fixed or variable charge.

  • Type I (fixed-charge). Cation has only one common charge. Includes Group 1 (always +1), Group 2 (always +2), Al (always +3), and a few transition metals fixed in their common ions: Zn (+2), Cd (+2), Ag (+1).
  • Type II (variable-charge). Cation has more than one common charge. Most transition metals fall here. Examples: Fe²⁺/Fe³⁺, Cu⁺/Cu²⁺, Sn²⁺/Sn⁴⁺, Pb²⁺/Pb⁴⁺.

7.3 Nomenclature for Binary Ionic Compounds

Type I — Fixed-Charge Cations

Naming rule: cation name (the metal's English name, unchanged) + anion root with -ide suffix. Example: NaCl = "sodium chloride." CaO = "calcium oxide." Mg₃N₂ = "magnesium nitride" (note the subscripts are not part of the name — charge balance does that work).

Common -ide endings:

  • F → fluoride; Cl → chloride; Br → bromide; I → iodide
  • O → oxide; S → sulfide; Se → selenide; N → nitride; P → phosphide; H → hydride

Type II — Variable-Charge Cations (Stock System)

Naming rule: cation name + Roman numeral in parentheses (= the charge of the metal in this compound) + anion root with -ide suffix. The Roman numeral is essential because the same elements can form more than one compound. FeCl₂ is "iron(II) chloride"; FeCl₃ is "iron(III) chloride."

EXAMPLE 7.1: Naming a Type II binary ionic compound

Name CuS.

Step 1 — Determine the anion charge: S is in Group 16, so S²⁻.

Step 2 — Determine the cation charge by balance: Cu must be +2 to balance one S²⁻.

Step 3 — Name: copper(II) sulfide.

Show solution

copper(II) sulfide.

EXAMPLE 7.2: From name to formula (Type II)

Write the formula for tin(IV) oxide.

Step 1 — Tin(IV) means Sn⁴⁺. Oxide is O²⁻.

Step 2 — Crisscross: Sn₂O₄.

Step 3 — Reduce: SnO₂.

Show solution

SnO₂.

7.4 Chemical Formulas and Names for Polyatomic-Ion Compounds

To write a formula for an ionic compound with a polyatomic ion, treat the polyatomic ion as a unit (consult Table 6.2 for charges) and balance charges. If more than one polyatomic ion is needed, enclose it in parentheses with the count as a subscript.

7.5 Nomenclature for Ionic Compounds Containing Polyatomic Ions

Naming rule: cation name (with Roman numeral if Type II) + polyatomic-ion name (no -ide suffix; use the ion's own name from Table 6.2). Examples: NaNO₃ = "sodium nitrate." K₂SO₄ = "potassium sulfate." Fe₂(SO₄)₃ = "iron(III) sulfate." (NH₄)₃PO₄ = "ammonium phosphate."

EXAMPLE 7.3: Compound with two polyatomic ions

Write the formula for ammonium dichromate.

Step 1 — Ammonium = NH₄⁺. Dichromate = Cr₂O₇²⁻.

Step 2 — Two NH₄⁺ are needed to balance one Cr₂O₇²⁻.

Step 3 — Formula: (NH₄)₂Cr₂O₇.

Show solution

(NH₄)₂Cr₂O₇.

7.7 Nomenclature for Acids

Acids are compounds that release H⁺ ions when dissolved in water. We will revisit the chemistry of acids later; for now we just need their names. Two patterns matter.

Binary Acids (Hydroacids)

A binary acid has the general formula H + nonmetal — in aqueous solution. Naming rule: hydro- + nonmetal root + -ic + acid. The (aq) label distinguishes the acidic solution from the molecular gas (HCl gas vs. HCl aqueous = hydrochloric acid).

7.8 Systematic Procedure for Nomenclature

To handle any compound you encounter, walk through this decision tree.

Figure 7.1 — Naming decision tree.

Alt text: A flowchart that takes a chemical formula as input and routes it through Yes/No decisions to the correct naming rule.

Description: A vertical flowchart drawn ~5 inches tall × 6 inches wide. Top node (rounded rectangle, gray): "Start: chemical formula given." An arrow leads down to the first decision diamond: "Does the formula start with H?" Yes (right): leads to "Aqueous?" → if yes → "Contains O? — yes → oxyacid (replace -ate→-ic acid, -ite→-ous acid); no → binary acid (hydro- root + -ic acid)." No (down) from the H question: "Is there a metal in the formula?" Yes → "Variable-charge metal? — yes → use Stock Roman numeral; no → use plain English name." Then continue: "Is there a polyatomic ion in the formula?" Yes → "Use the polyatomic-ion name (e.g., sulfate)." No → "Use the -ide ending on the second element." If the original "metal?" answer was No, branch right: "Two nonmetals → binary molecular: use Greek prefixes + -ide." Each terminal box is colored light blue. Use Arial 11 pt for box text and clear arrows.

Caption: Figure 7.1 — Decision tree for naming any compound covered in this chapter.

Worked Decision-Tree Examples

EXAMPLE 7.5: Decision-tree application

(a) Mg(OH)₂ — Starts with non-H → has a metal (Mg) → fixed charge → contains polyatomic ion (OH⁻) → "magnesium hydroxide."

(b) FeCl₃ — non-H start → metal (Fe) → variable charge → no polyatomic → -ide. Cation: each Cl is −1, total −3, so Fe = +3. Name: "iron(III) chloride."

(c) CO₂ — non-H start → no metal (C and O are nonmetals) → binary molecular → use Greek prefixes. Name: "carbon dioxide."

(d) HClO₂ (aq) — H start → aqueous → contains O → oxyacid → ClO₂⁻ is chlorite, so HClO₂ is "chlorous acid."

(e) HBr (aq) — H start → aqueous → no O → binary acid → "hydrobromic acid."

Multi-Concept Problems

1. A laboratory bottle contains a green crystalline solid labeled FeCl₃·6H₂O.

  1. What is the systematic chemical name (ignoring water of hydration)?
  2. The hydrate has 6 water molecules per formula unit; the prefix for hydrate naming is "hexa-," giving "iron(III) chloride hexahydrate." Confirm that the iron(III) charge is consistent with the chloride count.
Show solution
  1. Iron(III) chloride.
  2. 3 Cl⁻ × (−1) = −3; Fe must be +3 to balance → iron(III). ✓

2. Three compounds: H₂SO₄, H₂SO₃, H₂S.

  1. Name each.
  2. For each, identify whether it is a binary acid or oxyacid.
  3. Write the formulas of the conjugate base anions.
Show solution

H₂SO₄ — sulfuric acid (oxyacid); conjugate base SO₄²⁻ (sulfate). H₂SO₃ — sulfurous acid (oxyacid); SO₃²⁻ (sulfite). H₂S (aq) — hydrosulfuric acid (binary acid); S²⁻ (sulfide).

3. Write the systematic name and the formula for each ion or compound:

  1. bicarbonate (hydrogen carbonate),
  2. baking soda,
  3. lye (drain cleaner),
  4. limestone.
Show solution
  1. HCO₃⁻ (the polyatomic ion).
  2. Sodium bicarbonate, NaHCO₃.
  3. Sodium hydroxide, NaOH.
  4. Calcium carbonate, CaCO₃.
  5. CO and CO₂ are both binary molecular compounds of carbon and oxygen.

4. Why is one called "carbon monoxide" while the other is just "carbon dioxide"?

  1. Compare the bonding (recall Chapter 6).
Show solution
  1. The Greek prefix on the first element is dropped when it is "mono-," but the prefix on the second element is kept ("monoxide" for CO). For CO₂, the second is "dioxide."
  2. CO has a triple bond between C and O (very short, very strong); CO₂ has two double bonds (linear, nonpolar overall).

5. An anonymous formula in your notes reads "Cu₂SO₄."

  1. What is the name?
  2. Compare with CuSO₄ — name and Cu charge.
Show solution
  1. Cu must be +1 in Cu₂SO₄ to balance −2 of one SO₄²⁻ (since 2 Cu × (+1) = +2). Name: copper(I) sulfate.
  2. CuSO₄: Cu must be +2 → copper(II) sulfate (the more common one).

(A) FeS

(B) Fe₂S

(C) FeS₂

(D) Fe₂S₃

Answer Key: 1-C 2-C 3-C 4-B 5-B 6-B 7-C 8-B 9-D 10-A

Answer Key — Worked Rationales

1. C. Group-2 cations have only one common charge (+2), so no Roman numeral: simply "calcium chloride".

2. C. Iron has multiple oxidation states; in Fe₂O₃ each Fe is +3 → iron(III) oxide.

3. C. Phosphate is PO₄³⁻; balance with 3 Na⁺ → Na₃PO₄.

4. B. Sulfate is SO₄²⁻; balance with 2 NH₄⁺ → (NH₄)₂SO₄.

5. B. Two binary nonmetals: Greek prefixes for both → di-nitrogen pent-oxide → dinitrogen pentoxide.

6. B. Binary covalent: "mono-" is dropped from the first element → carbon tetrachloride.

7. C. Aqueous binary acid (H + halide): "hydro-" + halide-stem-"ic" + "acid" → hydrobromic acid.

8. B. Oxoacid from "sulfite" (SO₃²⁻): drop "-ite", add "-ous acid" → sulfurous acid.

9. D. "Chlorate" is ClO₃⁻; the corresponding acid is HClO₃ (chloric acid).

10. A. Iron(II) has charge +2; sulfide is S²⁻; balance gives 1:1 → FeS.

Concepts to Remember

  1. Different compound classes use different naming rules; classify before naming.
  2. Type I (fixed-charge) ionic: cation name + anion-root + -ide.
  3. Type II (variable-charge) ionic: add a Roman numeral in parentheses for the metal's charge.
  4. Polyatomic-ion compounds: keep the polyatomic ion's own name (no -ide suffix).
  5. Binary molecular: Greek prefixes count atoms; "mono-" is dropped on the first element.
  6. Binary acids (HX, aq): hydro- + nonmetal-root + -ic acid.
  7. Oxyacids: anion -ate → acid -ic; anion -ite → acid -ous.
  8. Charge balance always determines subscripts. The criss-cross trick + reduce gives the correct formula.
  9. Use the decision tree (Figure 7.1) to navigate any unfamiliar compound.

Key Terms Glossary

Binary acid (7.7) — An acid composed of hydrogen and one nonmetal in aqueous solution; named hydro- + nonmetal-root + -ic acid (e.g., HCl(aq) = hydrochloric acid).

Binary compound (7.1) — A compound containing exactly two elements.

Greek prefix (7.6) — A numerical prefix (mono-, di-, tri-, …) used in naming binary molecular compounds to indicate the number of atoms of an element.

Hydroacid (7.7) — Synonym for binary acid.

Oxyacid (7.7) — An acid composed of hydrogen, an oxy-anion, and oxygen; named by replacing the anion ending: -ate → -ic acid; -ite → -ous acid.

Stock system (7.3) — A naming convention that uses a Roman numeral in parentheses to specify the charge of a variable-charge metal cation (e.g., iron(III) chloride).

Type I ionic compound (7.2) — Binary ionic compound with a fixed-charge metal (Group 1, 2, Al, Ag, Zn, Cd).

Type II ionic compound (7.2) — Binary ionic compound with a variable-charge metal (most transition metals); requires a Roman numeral in the name.

Instructor Notes

Common Student Difficulties

  • Students name CuO as "copper oxide" without a Roman numeral. Insist on the Roman numeral for ANY transition metal not in the fixed-charge list.
  • Students confuse "-ate" anions with "-ide" anions. Drill with a list of -ate vs. -ide endings.
  • Students use Greek prefixes for ionic compounds ("dipotassium oxide" — wrong). Greek prefixes belong to MOLECULAR compounds only.
  • Conversely, students forget to use Greek prefixes for binary molecular compounds. Drill with NO, NO₂, N₂O, N₂O₃, N₂O₅.

Recommended In-Class Activities

  • Decision-tree posters: each pair receives a list of formulas and walks through Figure 7.1 to name them, displaying their work on a large poster.
  • Naming relay: each student is given one formula or one name and must produce the other; cards rotate.
  • Polyatomic-ion karaoke: print a song-lyric mnemonic; rote learning supported by the music.

Lab Connections

  • Identification of household products: bleach (NaClO), washing soda (Na₂CO₃), milk of magnesia (Mg(OH)₂), aspirin (HC₉H₇O₄). Students name each and classify.
